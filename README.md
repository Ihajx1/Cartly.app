Cartly - Shopping Cart Mobile App

Cartly is a mobile shopping cart application developed using Android Studio.
The app allows users to browse products, add them to a cart, and manage their shopping experience in a simple and efficient way.

Features:
- User Registration and Login
- Browse Products
- Add Items to Cart
- Remove Items from Cart
- Local Database using SQLite


Technologies Used:
- Java
- Android Studio
- SQLite Database
- XML for UI Design
